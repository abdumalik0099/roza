## 🏗️ APK qurish yo'llari

### 1️⃣ ENG OSON: GitHub Actions (bepul, 5 daqiqa)

1. GitHub.com da **yangi repository** yarating
2. Barcha fayllarni yuklang (yoki `git push` qiling)
3. **Actions** bo'limiga o'ting
4. **Build APK** workflow ni ishga tushiring (Run workflow)
5. Tugagach **Artifacts** bo'limidan APK ni yuklab oling ✅

### 2️⃣ Android Studio (kompyuterda)

**Talab:** Android Studio (bepul yuklab olish: developer.android.com/studio)

1. Android Studio ni oching
2. **File → Open** → bu papkani tanlang
3. SDK yuklanishini kuting (~5 daqiqa)
4. **Build → Build Bundle(s)/APK(s) → Build APK(s)**
5. `app/build/outputs/apk/debug/app-debug.apk` faylini oling ✅

### 3️⃣ Online APK Builder (brauzor orqali)

1. **appetize.io** yoki **buildroid.com** dan foydalaning
2. Loyiha ZIP faylini yuklang
3. APK ni yuklab oling

## 📲 Telefonga o'rnatish

APK faylni olgach:
1. Telefoningizga nusxalang
2. **Sozlamalar → Xavfsizlik → Noma'lum manbalar** ni yoqing
3. APK faylni bosing va o'rnating

> ⚠️ Google Play dan tashqaridan o'rnatish uchun "Noma'lum manbalar"ni ruxsat berish kerak
